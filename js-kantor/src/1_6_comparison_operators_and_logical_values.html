Операторы сравнения и логические значения

  • a > b, a < b
  • a >= b, a <= b
  • a == b
  • a != b

Логические значения

Ну тут - true | false. Можно присвоить явно или как результат сравнения:

var a = true;
var b = 3 > 4;

Сравнение строк

Строки сравниваются побуквенно (лексикографическое). При этом происходит сравнение по численным кодам символов Unicode.

"Б" > "А" // true
"а" > "Я" // true

Сравнение разных типов

При сравнении значений разных типов используется числовое преобразование.

"2" > 1 // true, 2 > 1
"01" == 1 // true, 1 == 1
false == 0 // true, false = 0
true == 1 // true, true = 1

Строгое равенство

0 == false // true: проблема, так как равенство не отличает 0 от false
"" == false // true: проблема, так как равенство не отличает пустую строку от false

Это следствие того, что операнды разных типов преобразовались к числу

Для проверки равенства без преобразования типов используется строгое равенство и неравенство: === и !==

Сравнение с null и undefined

  • Значения null и undefined равны друг другу, и не равны чему бы то ни было еще.
  • При преобразовании в число null - становится 0, а undefined - NaN.

  • Некорретный результат сравнения null с 0

    null > 0 // false
    null == 0 // false - тут несмотря на то, что null приводится к 0, он не может равняться чему либо кроме себя
    null >= 0 // true - здесь честно приводит к числу 0

  • Несравнимый undefined

    undefined > 0 // false
    undefined < 0 // false - приводится к NaN
    undefined == 0 // false - не может равняться ничему, кроме себя
