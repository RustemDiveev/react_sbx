Переменные

Ограничения на имена

  • Имя может состоять из бука, символов, $ и _
  • Первый символ не должен быть цифрой
  • Не должно быть зарегистрированным словом: var, class, return, export, etc...

Регистр переменной тоже ролляет - jopa и JoPa - разные переменные.

Можно использовать русские буквы, но лучше всегда - английские.

Правила именования переменных

  • Никакого транслита, только английский
  • Использовать короткие имена только для переменных "местного значения"
  • Использовать camelCase
  • Имя переменной должно соответствовать хранимым в ней данным

Например, переменная для названия нашей планеты - myPlanet: var myPlanet = "Земля";

Например, имя посетителя сайта - visitorName: var visitorName = "Петюня";

Типы данных

  • Число "number"

    var n = 12.345;

    Специальные значения - Infinity (бесконечность) и NaN (ошибка вычислений)

    var inf = 1 / 0; //infinity
    var nan = "string" * 2; //NaN

  • Строка "string"

  • Булево значение "boolean"

    Два значения: true | false

  • Специальное значение "null"

    Cпециальное значение со смыслом "ничего" или "значение неизвестно"

  • Специальное значение "undefined"

    Cпециальное значение со смыслом "значение не присвоено"

  • Объекты "object"

    Верхние типы - примитивные; этот - отдельный, для коллекций данных и сложных сущностей.

Есть оператор typeof, возвращающий тип аргумента:

typeof x
typeof(x)

Возвращает строку с типом данных, есть вот такие приколы (остальное поведение логично):

typeof null == "object"
