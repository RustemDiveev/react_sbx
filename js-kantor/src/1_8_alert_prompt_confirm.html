Взаимодействие с пользователем: alert, prompt, confirm

alert(text)

Выводит на экран окно с сообщением и приостанавливает выполнение скрипта, пока пользователь не нажмет "ОК".

prompt(title, default)

Выводит модальное окно с заголовком title, полем для ввода текста, заполненным строкой по умолчанию default,
и кнопками OK/CANCEL. Возвращает то, что ввел пользователь, если было нажато OK, и null - если CANCEL.
Лучше всегда указывать default, иначе, например IE, вставит undefined.

confirm(text)

Выводит окно с text и кнопками OK/CANCEL. Вернет true - если нажато OK, false - если CANCEL.

Особенности встроенных функций

Где выводится окно - решает браузер, и на это никак не повлиять. Эти функции - очень простые.
