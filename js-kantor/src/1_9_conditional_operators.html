Условные операторы: if, '?'

Оператор if

Оператор получает условие, вычисляет его, и если результат true - то выполняет команду.
Если нужно выполнить более одной команды, то они разделяются фигурными скобками.
Рекомендуется использовать их всегда, так как это улучшает читабельность кода.

Преобразование к логическому типу

  • false: - 0, "", null, undefined, NaN
  • true: - все остальное

Несколько условий, else, else if - пример

if (year < 2022) {
 alert("Too old!");
} else if (year < 2022) {
 alert("Too late!");
} else {
 alert("Exactly that year");
}

Оператор вопросительный знак ?

условие ? если true, то значение 1 : если false, то значение 2

Можно делать цепочки:

var message = (age < 3) ? "Hello, baby!" :
  (age < 18) ? "Hello" :
  (age < 100) ? "Greetings" : "Are you dead?";
